# Taller GNC

Aplicación web (PWA) para la gestión de clientes de talleres de GNC: turnos de
Oblea y Prueba Hidráulica, datos de clientes y vehículos, y lectura automática
de datos desde fotos con IA.

El código de la aplicación está en [`taller-gnc-web/`](taller-gnc-web/).

Para instrucciones de instalación y despliegue, ver
[`taller-gnc-web/LEEME.md`](taller-gnc-web/LEEME.md).
